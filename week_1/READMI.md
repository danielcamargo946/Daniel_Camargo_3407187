📝 Descripción
Este repositorio contiene el desarrollo del módulo de JavaScript para el proyecto AutoSave. Se enfoca en la creación de una ficha técnica de activos digitales integrando la plataforma Kraken, aplicando estándares modernos de programación y lógica de desarrollo de software.

🛠️ Competencias en JavaScript
Sintaxis Moderna: Uso de ES2023 para la manipulación de datos y visualización en consola.

Lógica de Dominio: Implementación de una estructura de "Ficha de Presentación" aplicada al trading de criptomonedas.

Manejo de Tipos de Datos: Uso de Strings, Numbers para cálculos financieros y Booleans para estados de validación.

Operaciones Aritméticas: Cálculos automáticos de inversión y conversiones de activos (ej. BTC a USD).

🚀 Estructura del Script (script.js)
El código principal realiza las siguientes funciones dentro del ecosistema Fintech:

Identificación de Entidad: Registro de activos como Bitcoin (XBT) con identificadores únicos.

Cálculos Dinámicos: Procesamiento de valores de inversión mediante expresiones aritméticas directas.

Estados de Sistema: Validación booleana de estados del activo (verificación en Kraken, disponibilidad de Staking).

⚙️ Requisitos y Ejecución
Para ejecutar este proyecto en tu entorno local (macOS), necesitas tener instalado Node.js.

Verificar instalación de Node:

Bash
node -v
Ejecutar la ficha de dominio:

Bash
node starter/script.js
